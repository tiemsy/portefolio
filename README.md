# Portfolio MTC — CSS custom sans Bootstrap

Site portfolio PHP moderne, responsive et dockerisé.

## Pages incluses

- `index.php`
- `pages/skills.php`
- `pages/experiences.php`
- `pages/realizations.php`
- `pages/contact.php`

## Lancer avec Docker

```bash
docker compose up --build
```

Puis ouvrir :

```text
http://localhost:8081
```

## Arrêter

```bash
docker compose down
```

## Images et CV

Place tes images dans `images/` et ton CV dans `pdf/cv_mohamed_camara.pdf`.
